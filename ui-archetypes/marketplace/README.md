# Marketplace archetype

**Inspired by**: Mercado Libre, Etsy, Airbnb.

Grid-first layouts, social cards, promotional rails, fast filters. The mobile
experience showcases discovery and category navigation rather than dense
operational data.

- **Density**: comfortable
- **Nav**: top bar with category rail
- **Cards**: social (image, title, badge, price, secondary action)
- **Animation**: snappy
- **Best for**: marketplace, restaurant ordering, business networks, ecommerce

Activate from **Admin → UI Mode** or set
`vforge-archetype=marketplace` cookie.
