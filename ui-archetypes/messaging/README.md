# Messaging archetype

**Inspired by**: WhatsApp, iMessage, Telegram.

Conversation-first interface. List of threads as the home shell, with a sticky
composer, read receipts, lightweight cards and clear separation of incoming /
outgoing bubbles.

- **Density**: comfortable
- **Nav**: bottom-sheet on mobile, rail on desktop
- **Cards**: social, thread-style
- **Animation**: snappy
- **Best for**: school communications, business networks, community apps,
  internal comms, support inboxes

Activate from **Admin → UI Mode** or set
`vforge-archetype=messaging` cookie.
