# Maps module

Google Maps wrapper with sensible defaults and demo fallback.

- Components: `<MapView>`, `<MapMarker>`, `<MapAutocomplete>`
- Geocoding + reverse geocoding helpers
- Reads `NEXT_PUBLIC_GOOGLE_MAPS_API_KEY`
- Falls back to a stylized static map when the key is missing
- Used by real-estate, rides, security-inspections, business-network
